# Chapter 4: The Geometry of Data 
## Overview
In this chapter, I explore the ways in which data can be interpreted geometrically using vectors and linear algebra.
Through creating vector operations, I explore the role of distance, direction, and magnitude in how we measure relationships in data.

## Core Question
How can linear algebra concepts like vectors and matrices help us analyze relationships within real-world datasets?

## Data Source 
This project used a movie data set from Kaggle consisting of 3886 films scraped from Hydra Movies full collection of movies.
https://www.kaggle.com/datasets/kayscrapes/movie-dataset

## My Projects 
* [**01_vector_operations.ipynb**](./01_vector_operations.ipynb): In this project, I represent cereal nutrition data as vectors and perform vector operations such as cosine similarity and vector subtraction. These operations are used to explore relationships between nutritional variables and compare specific cereals.

* [**02_matrix_operations.ipynb**](./02_matrix_operations.ipynb): In this project, I represent movie data as a matrix where rows correspond to films and columns represent numerical features. I perform basic matrix operations such as calculating row and column statistics, identifying extreme values, and computing matrix transposes to explore the structure of the dataset.
